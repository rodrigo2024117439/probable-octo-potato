# Deep Sea Mining - Modelo de Dados (Stage 1)

**Descrição:**
Este package contém o modelo de dados para o jogo "Deep Sea Mining".
O objetivo destas classes é gerir o estado do jogo, recursos, veículos (Navio e Drones) e a geração dos mapas (Superfície, Fosso e Fundo do Mar).
Não inclui qualquer lógica de interface gráfica, garantindo a separação de responsabilidades.
